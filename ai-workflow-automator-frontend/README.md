# AI Workflow Automator — Frontend

React frontend for the AI Workflow Automator backend.

## Setup

```bash
npm install
```

Create a `.env` file:
```
REACT_APP_API_URL=http://localhost:3000/api
```

## Run

```bash
npm start
```

Opens on http://localhost:3001 (backend runs on 3000).

## Features
- Dashboard with stats and recent executions
- Create, enable/disable, delete workflows
- Run workflows manually
- View execution history and AI results
